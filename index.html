Project ini dibuat untuk memenuhi tugas course mingguan, berupa kalkulator sederhana untuk menghitung luas dan keliling segitiga. menggunakan teknoligi html css dan javascript.

K = S + S + S

  • K = Keliling
  • S = Sisi

L = 1/2 x A x T


  • L = Luas
  • A = Alas
  • T = Tinggi
